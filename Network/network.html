Redes de Computadores

As redes de computadores são sistemas de comunicação que permitem a troca de informações e recursos entre dispositivos interconectados, facilitando a comunicação e o compartilhamento de dados em escala local e global.

OSI e TCP/IP

O modelo OSI e o protocolo TCP, são dois conceitos fundamentais em redes de computadores. Eles desempenham papéis distintos, mas complementares, na comunicação e no intercâmbio de dados entre dispositivos em uma rede.
O modelo OSI é amplamente utilizado como referência para entender e projetar redes de comunicação. O TCP é um protocolo de transporte confiável que opera na camada de transporte do modelo OSI. Ele fornece controle de fluxo, controle de congestionamento e garantia de entrega dos pacotes de dados. Ele também lida com a retransmissão de pacotes perdidos ou corrompidos.

Veja cada camada aqui

➪
Camada Física:

 Lida com a transmissão de bits brutos por meio de meios físicos, como cabos, sinais elétricos ou ondas de rádio.

Camada de Enlace de Dados:

 Fornece um fluxo confiável de dados por meio de uma conexão física. Lida com a detecção e correção de erros, bem como o controle de acesso ao meio físico.

Camada de Rede:

 Responsável pelo roteamento dos pacotes de dados pela rede. Determina o caminho mais eficiente para enviar os dados de um nó para outro.

Camada de Transporte:

 Lida com o controle de fluxo e a segmentação dos dados em pacotes menores para uma transmissão eficiente. O protocolo TCP opera nesta camada.

Camada de Sessão:

 Estabelece, gerencia e encerra as conexões entre as aplicações em diferentes dispositivos. Também oferece serviços de controle de diálogo.

Camada de Apresentação:

 Lida com a representação dos dados, incluindo codificação, compressão e criptografia. Também trata de questões de formatação e tradução de dados entre diferentes sistemas.

Camada de Aplicação:

 É a camada mais próxima dos usuários finais e fornece interfaces para serviços de rede, como e-mail, transferência de arquivos e acesso remoto.
