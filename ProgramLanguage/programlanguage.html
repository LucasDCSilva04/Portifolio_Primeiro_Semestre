Linguagem de Programação

Existem muitas linguagens de programação disponíveis atualmente, cada uma com suas características e finalidades específicas. Aqui estão algumas das linguagens de programação mais populares:

Java

Java: É uma linguagem de programação orientada a objetos popular, usada principalmente para desenvolvimento de aplicativos corporativos, aplicativos Android e sistemas em larga escala. É conhecida por sua portabilidade e segurança.

Em java foram feitos projetos muito interessantes, no meu repositório, há exercícios feitos em sala de aula com o professor Cainã, há também o "JogoDeTiro", que eu estou desenvolvendo ainda! Também há exercícios que foram propostos pela plataforma da DIO.me, sobre Try Catch, Pilares da Programação Orientada a Objetos e muito mais. Confira abaixo!

JogoDeTiro.java

Neste jogo, ja deixo claro no início: ChatGPT me ajudou muito, logo, não foi totalmente desenvolvido por mim, porém, este projeto me ajudou demais a entender como que "conversa" em Java, por exemplo, vi como puxar arquivos, imagens, sons, etc, e exibi-la em uma janela, também aprendi como abrir uma janela, aprendi como puxar o estado de uma tecla, aprendi a montar uma animação bem básica. Como está bem simples, não precisei usar Thread, mas quando for implementado os tiros, as músicas de fundo, os inimigos, entre outros, com certeza vou usá-lo. ConversorBinario.java

Esse projeto começou sem querer, o professor Cainã estava comentando sobre conversões de variáveis, como de 'int' -> 'double', logo me veio a ideia de fazer um conversor, de números binários, para ASCII e também para decimal. Parece simples, mas nesse projeto utilizei o Scanner.in, que resgata o valor digitado no console, também usei o Try-Catch, para tratar excessões, no exemplo do vídeo a seguir, escrevi uma letra de propósito, e letra não é binária, assim como qualquer valor além de '0' e '1' também não é, logo, tratei esse possível erro que o cliente pode cometer. Logico que dá para melhorar esse projeto, onde eu faço as conversões contrárias, mas isso fica para um projeto mais moderno no futuro.

Estoque.java

Essa atividade foi muito top, gostei muito, pois chega próximo a um sistema de estoque, e conforme vamos aprendendo a programar, vem na imaginação muitos detalhes que automatiza o sistema, por exemplo, neste projeto, ja mostra no início o que você deseja fazer, ali você navega pelo sistema, adiciona produtos, remove, edita, entre outros. Neste projeto, aprendi principalmente o Scanner.In, utilizamos o loop Do-While, utilizamos o Switch-Case, que otimiza muito o código ao invés de utilizar vários If's, logo, dentro de cada Case chamamos os métodos, que tem cada um sua função, seja cadastrar, adicionar, etc. Aprendemos aí sobre construtores, métodos, modos estáticos, públicos, privados, Try-Catch, como funciona o Return, no toString, aprendemos como tabular o texto, quebrar linha e etc. Bem completinho

Matriz.java

Na aula de Matriz, aprendi melhor sobre Arrays, no exemplo a seguir, pode conferir que utilizamos o Array bidimensional, porém, pode se construir arrays com infinitas dimensões. Neste exercício, aprendi a manipular a matriz, la no console mostra os maiores valores resgatados, a média de valores, e até a rotação desta matriz. Para isso, foi utilizado o laço for, que otimiza demais o código, por conta de ser um loop. Ele resgata valor por valor, porém, "Por que um For dentro de outro For?". Isso acontece por que o primeiro For define a linha da matriz, e o segundo For define a coluna da matriz, logo, o segundo For percorre a primeira linha, coluna por coluna, depois passa para a proxima linha sucessivamente.

ScreenProtector.java

Esta aula foi confusa quando começou a aparecer coisas novas, além das que já tinham nos outros projetos, porém, conforme o professor foi explicando, ajudou demais a entender sobre herança, onde, no exemplo, Square herda de Rectangle que herda de Shape, da mesma forma com Shape <- Elipse <- Circle, mas não se limita aí! Foi passado também como trabalhar com Thread, que nada mais é: efetuar varias ações ao mesmo tempo, como a música de fundo junto com o movimento do personagem. Vimos sobre modificadores, sobre polimorfismo, sobre como faz uma sobrecarga e por que usa-la, foi passado também o tratamento de colisão, como definir parametros, extender de outras classes, importar bibliotecas, como utilizar o random, como puxar o estado de uma tecla, e muito mais.

Python

Python: É uma linguagem de programação de alto nível, fácil de aprender e com uma sintaxe legível. É usada em uma ampla gama de aplicações, desde desenvolvimento web e científico até automação de tarefas e inteligência artificial.

Python sofre muito bullying, muitos dizem que não é uma linguagem de programação, por conta de suas diversas bibliotecas, porém entra aqui na lista sim! Em python, os primeiros exercícios propostos foi no Google Colab, onde vimos tabelas, aprendemos a exibir dados, tirar média, mediana, moda, porém esse assunto está muito ligado a Ciência de Dados, logo, na página de Ciência de Dados, você irá encontrar mais detalhes.

JavaScript

JavaScript: É uma linguagem de programação voltada para o desenvolvimento web. É amplamente utilizada para criar interações dinâmicas em sites e aplicativos web, além de ser usada também em servidores (Node.js) e em desenvolvimento de aplicativos móveis.

C

C: É uma linguagem de programação de propósito geral e de alto desempenho. É amplamente utilizada no desenvolvimento de sistemas operacionais, jogos, software embarcado e aplicativos que exigem eficiência e controle de hardware.

A linguagem C é da família do JavaScript, não se compara, mas é como se fosse a mesma lingua com alguns sotaques, só que utilizada no Arduino, por exemplo. Na página de automação será detalhado com meus projetos feitos em linguagem C.
